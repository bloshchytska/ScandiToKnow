# ScandiToKnow

## Story

### Goal
To make an image classifier


### Methods used
* Image hashing, ImagePHash
* Hamming and Cartesian distance
* (last version) with tensorflow

### Flow

1. Take a photo.
2. Recognise image.
3. Display the result in the form of the image found and the text describing it.
